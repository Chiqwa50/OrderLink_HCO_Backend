-   npm أو yarn\

    به\

------------------------------------------------------------------------

## 2. معماريّة المشروع ومكوّناته

-   **Frontend:** Next.js (React) + Tailwind CSS\
-   **Backend (API):** Node.js + Express + Prisma ORM\
-   **Database:** PostgreSQL (Supabase / Neon / Railway)\
-   **Auth:** JWT (JSON Web Tokens) - بسيط لتسجيل الدخول/التحقق بالهاتف\
-   **PDF generation:** Puppeteer أو pdfmake أو html-pdf (يمكن تشغيلها
    كخدمة على الـ API)\
-   **Realtime (اختياري لاحقًا):** Socket.IO أو WebSockets أو Pusher\
-   **Hosting:** Vercel (Frontend) + Render/Railway (Backend) + Supabase
    (DB & Auth optional)
